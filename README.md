# vibe-learn

**vibe-learn** is an Obsidian template built on top of [Templater](https://github.com/SilentVoid13/Templater) to shift from passive knowledge-base management to active engagement with own study materials and help in knowledge retention.

## 🛠️ Key Features

| Feature | Description |
| :--- | :--- |
| **Auto-Metadata** | Automatic datetime stamps and dynamic tag creation based on the target concept. |
| **Focus Tools** | Integrated **Time Tracker** and **Pomodoro Status Bar** to keep you in the flow state. |
| **Visual Progress** | Uses the **Heatmap Tracker** plugin to visualize your "learning streaks." |
| **Data-Driven** | Leverages **Dataview** to create automated dashboards of your study habits. |
| **Self-Quizzing** | Integrated with **Quiz Generator** plugins to turn your notes into active recall sessions. |

---

## Getting Started

### Prerequisites
To get the full "vibe-learn" experience, you'll need the following community plugins installed:
1.  **Dataview** (For indexing and dashboards)
2.  **Templater** (To run the reactive prompts)
3.  **Heatmap Tracker** (For the "Duolingo" style progress view)
4.  **Spaced Repetition** or **Quiz Generator** (For active recall)
